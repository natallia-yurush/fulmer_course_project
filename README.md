# fulmer_course_project

Разработка системы прогнозирования платежеспособности предприятия на основе модели фулмера

Приложение выполнено в архитектуре клиент-сервер с многопоточным сервером с организацией взаимодействия с базой данных на языке Java.

Уровни архитектуры: Серверное приложение реализовано в виде консольного приложения. Клиентское
приложение — оконное приложение с использованием библиотеки пользовательского
интерфейса JavaFX.
